
## Prompt Templates
LLM-RE_Prompt_Templates.docx file contains prompt templates for Inference and CoT generation for all three datasets.

## Datasets:
### Semeval 2010 Task-8 Dataset
    * Path: LLM-RE/semeval2010-task8-dataset
    * This folder contains Semeval 2010 Task-8 Dataset and their results of LLM-RE framework.
    
### TACRED
    * Path: LLM-RE/tacred
    * This folder contains results of LLM-RE framework for TACRED dataset.
    * TACRED dataset is not shared due to Restrictive copyrights.
    
### RE-TACRED
    * Path: LLM-RE/re-tacred
    * This folder contains results of LLM-RE framework for RE-TACRED dataset.
    * RE-TACRED dataset is not shared due to Restrictive copyrights.
    

## LLM-RE Code
### Semeval 2010 Task-8 Dataset
* Code for Inference is available at: [https://www.kaggle.com/code/kartikdetroja/llm-re-semeval2010-task8](https://www.kaggle.com/code/kartikdetroja/llm-re-semeval2010-task8)
* Same file is available in the repository: llm-re-semeval2010-task8.ipynb

###  TACRED Dataset
* Code for Inference is available at: [https://www.kaggle.com/code/kartikdetroja/llm-re-semeval2010-task8](https://www.kaggle.com/code/kartikdetroja/llm-re-tacred)
* Same file is available in the repository: llm-re-tacred.ipynb

### RE-TACRED Dataset
* Code for Inference is available at: [https://www.kaggle.com/code/kartikdetroja/llm-re-semeval2010-task8](https://www.kaggle.com/code/kartikdetroja/llm-re-retacred)
* Same file is available in the repository: llm-re-retacred.ipynb

Results of LLM-RE Framework are available in results folder of respective datasets. Post-processing and result-analysis scripts to compute F1-score are also availalbe in result folder of respective datasets.
